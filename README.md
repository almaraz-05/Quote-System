# Quote Management System

A database-driven PHP web application that allows sales associates to create, manage, approve, and convert customer quotes into purchase orders.

## Features
- Sales associate login system
- Create and edit customer quotes
- Add line items to quotes
- Track quote status (open, finalized, sanctioned, ordered)
- Convert sanctioned quotes into purchase orders
- Persistent storage using MariaDB
